== Description ==

Product extra-options with dynamic pricing for WooCommcerce. CCF add cumulative custom fields with live calculation to any WooCommerce products.

Price is live updated according preset values and the purchase funnel is ready : custom data is displayed in all steps of the order process. 

** Regular version ** : 2 cumulative fields with 14 types of options available.
** Pro version ** : n cumulative fields with 14 types of options available and conditionals (set your own maximum, mininum is 2).

*   Text
*   Select dropdown => from 2 to 6 select options.
*   Checkboxes => from 1 to 4 checkboxes.
*   Textarea field
*   Radio buttons => from 2 to 4 radio buttons.
*   Number field
*   Images swap => from 2 to 4 Images.
*   E-mail field
*   Buttons swap => from 3 to 6 buttons.
*   Color field
*   Calculation and custom formula => from 1 to 2 calcul inputs.
*   Url field
*   Ajax upload field => 100 file mime types available. [front-upload in uploads folder]
*   Phone field

== Requirements ==

*    Wordpress installed (online or local web server).
*    WooCommerce installed and activated (you can skip store settings). 
*    At least WooCommerce version 8.4.0 required.
*    Product regular price must be fixed before to start.

== Installation ==

1. Unzip 'calculated-custom-fields' and paste its contents into the folder `/wp-content/plugins/`
2. Choose or create a product
3. Edit the product and select "calculated custom fields" in the 'product data' panel. 
4. For each field, choose the type of options and the corresponding information.
5. Click on "Save and display option" and go on the product page or refresh it. 
6. The settings tab allows you to define some text strings on the product page and the number of custom fields (pro version).
